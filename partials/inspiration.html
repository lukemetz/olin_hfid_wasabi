Inspiration

Inspiration can be found everywhere! For a more in-depth discussion click the links bellow:

Code Academy By Luke Metz
Google Maps for Mobile By Morgan Zhu
Day One By Michael Ninh
Calendar By Keely Haverstock

  • Outlook/Google Calendar: Potential set-up to display scheduling and availability.
  • AdAstra: The existing system. A lot of it is awful but some features, such as images of the room, are useful.
  • Going to a room and looking: How things were done and the most natural way to get information on a room.
  • Room designing/CAD software: These programs allow users to explore a room virtually.
  • Pandora: Uses a tagging system to group similar types of music.
  • Youtube: Provides recommendations for users. Also a potential implementation of time-based browsing a la scrubbing through a video.
  • Facebook: Allows users to continue scrolling through a timeline.
  • Amazon: Browse under existing categories or search and specify attributes â€”> easy to transfer from browsing to buying.
  • Gmail: Central view of useful information. Shows you everything, quick interface to write email and be done. Heads up notifications.
  • Guidebook app: Shows information for just one thing at a time. Hierichal browsing for information which may be applied to sorting rooms.
  • Google maps (mobile): Search for things and see how they are laid out spatially and quickly pick location and navigate to it. Also provides zooming from abstract to actual photos.
  • Chrome webstore: Nice type ahead search to suggest apps for the user.
  • Gmail Android interface: Slide to delete and provides a temporary undo button rather than prompting.
  • Github: Shows what happened with time. Flow back in time. Nice tutorial at the top to show how to use it.
  • Gmail: Uses an overlay tutorial which can be hidden or revealed at any time.
  • iOS7: Provides a walkthrough to both set-up the phone and highlight new features which may be frequently used.
  • Post-it notes: Easy to reorder information, can give nice physical way of looking at information. Allows users to define their own organization scheme.
  • Olin Launchpad: Search feature filters things automatically.
  • Elevator buttons: Clearly labeled buttons -> one-to-one mapping of information and function.
  • whenisgood: Scheduling app with very fast interface to quickly select lots of times. Allows users to quickly determine what times are available for everyone.
  • Doodle: Horizontal view of time vs vertical. Shows other users using the system well.
  • iCals: Automatically puts event data into a calendar and provides reminders.
  • CodeAcademy: Immediate feedback while typing and when editing code. Interactive overlay to provide additional explanations.
  • Galaxy S3: Includes a small indicator LED that changes colours to show new messages.
  • Mobile Gmail: Easy accessing and reading emails from several different email accounts on the go.
  • Audacity: Music editor which shows multiple tracks on top of each other. A potential implementation to display availability of multiple rooms.
  • Evernote: Syncs collaborative notes for multiple users. Alerts users to changes.
  • Vending machines: Direct mapping of function and outcome.
  • Venmo: Very simple interface for sending money. Provides feedback to both senders and receivers and allows a transaction to be initiated from either party (payments and charges).
